\begin{problem}{โจทย์สุดง่าย (มั้ง)}{standard input}{standard output}{0.25 seconds}{256 megabytes}{100}

นักเรียนคนหนึ่งนั่งอยู่ในห้องเรียนวิชาคณิตศาสตร์ แต่เมื่อคืนเขาทำงานหนักเกินไปหน่อยเลยนอนดึกทำให้ง่วงนอนจนหลับในห้องเรียน แต่เนื่องจากเขานั่งอยู่ในแถวหน้าๆ ทำให้อาจารย์เห็นเข้า อาจารย์จึงให้ยืนขึ้นแล้วตอบคำถามที่เขียนอยู่บนกระดาน (เนื้อเรื่องที่คุ้นเคยจาก TUMSO18)

ให้ $x$ เป็นจำนวนเต็ม และ \[f(x) = x^3 - 6x^2 + 11x - 6\]
และ \[g(x) = x^2 - 3x + 2\]
จงหาค่าของ \[\frac{f(x)}{g(x)}\] โดยหากไม่สามารถหาค่าได้ให้ตอบ\begin{no}
    NO
\end{no}

หมายเหตุ : คำตอบอาจมีขนาดใหญ่ให้ตอบเป็นเศษที่เกิดจากการหารคำตอบด้วย $10^9 + 7$

\InputFile

บรรทัดที่ $1$ รับจำนวนเต็ม $Q$ แสดงถึงจำนวนคำถาม $1 \leq Q \leq 10^5$

บรรทัดที่ $2$ ถึง $Q + 1$ รับจำนวนเต็ม $x_i$ $-10^{18} \leq x \leq 10^{18}$

\OutputFile
มีจำนวน $Q$ บรรทัด ซึ่งบรรทัดที่ $i$ แสดงถึงเศษจากการหารคำตอบของคำถามที่ $i$ ด้วย $10^9+7$

\Scoring
ชุดทดสอบจะถูกแบ่งเป็น 2 ชุด จะได้คะแนนในแต่ละชุดก็ต่อเมื่อโปรแกรมให้ผลลัพธ์ถูกต้องในชุดทดสอบย่อยทั้งหมด

\begin{description}

\item[ชุดที่ 1 (31 คะแนน)] จะมี $1 \leq Q \leq 10^2, -10^3 \leq x \leq 10^3$
\item[ชุดที่ 2 (69 คะแนน)] ไม่มีเงื่อนไขเพิ่มเติม 

\end{description}

\end{problem}
